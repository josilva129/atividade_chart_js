# 📊 Dashboard com Chart.js

Este projeto é uma página HTML simples que utiliza a biblioteca **Chart.js** para criar uma dashboard com dois gráficos:

* Gráfico de linha (Temperatura x Umidade ao longo do dia)
* Gráfico de barras (Média mensal de Temperatura e Umidade)
